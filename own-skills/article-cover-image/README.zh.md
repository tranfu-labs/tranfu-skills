# 文章封面图生成

给一个标题或整篇文章, 直接出一张 16:9 横版封面图, 主色 #E63A46, MiSans 风格标题, 材质哑光克制, 拒绝信息图化。

## 什么时候用它

**只有标题**

我手上就一句标题 / 一个观点, 想让 skill 直接给张封面, 别先写一大段分析。

**标题加正文**

我文章草稿写完了, 需要一张横版封面, 用在公众号 / 视频 / PPT 首页 / 网站 Banner 都行。

**只要 prompt**

我想拿到生成 prompt 自己去别的模型跑, 不用 skill 出图。skill 切到文字模式, 只给主 Prompt + Negative Prompt。

**无字图**

我自己上标题, 只要背景图, 留一块干净的文字安全区。

**换风格**

我不想每次都是深蓝科技感, 指定杂志风 / 电影感 / 哑光立体图标, skill 就从八种风格池里换一条路走。

**不接**

- 图标 / 黑白线性符号 → **black-line-icon-style**
- 文中插图 (不是封面那张) → **post-illustration-images**
- 信息图 / 知识卡片 / 对比表 / 流程长图 → 信息图 workflow
- 品牌正式 Logo 定稿 → Logo 设计 workflow
- 只要文案 / 非视觉分析 → 写作 workflow

## 它会产出什么

**默认直接出图, 不先写方案** —— 除非你明确说「只要 prompt / 不生成图 / 先分析」, skill 才切到文字模式。

- **封面图**: 一张 16:9 横版图, 主色 #E63A46 明确出现, MiSans 风格中文标题, 哑光低反射材质
- **标题压缩**: 长标题会被压成不超过 12 字的封面短标题 (例:「为什么复杂任务的默认工作流应该写在 AGENTS.md 而不是某个 Skill 里」→「AGENTS.md 优先」)
- **风格自动路由**: 根据标题关键词, 从八种风格里挑一种 —— 极简杂志 / 现代产品 UI / 电影感隐喻 / 潮流插画 / 商业 Editorial / 哑光立体图标 / 抽象艺术 / 科技未来
- **文字模式**: 明说「只要 prompt」时, 只输出主 Prompt + Negative Prompt, 不调用图像生成
- **绝不会做**: 生成信息图 / VS 对比图 / 多主体并列 / 塑料玩具感 3D / 副标题加长文案

## 前置条件与边界

**前置**

需要能调用图像生成能力 (不支持时只能走「只要 prompt」模式); 至少给一个标题, 或正文里能提炼出主题。

**相邻 skill 分工**

| 想要什么 | 交给 |
|---|---|
| 图标 / 黑白线性符号 | **black-line-icon-style** |
| 文中插图 (非封面) | **post-illustration-images** |
| 信息图 / 知识卡片 / 流程长图 | 信息图 workflow |
| 品牌正式 Logo | Logo 设计 workflow |

**不接的场景**

- 既没标题, 正文也提炼不出主题 → skill 停下问你要标题
- 只要文章文案 / 普通写作 → 走写作 workflow
- 只要非视觉分析 / 通用提示词 → 不触发

**微妙边界**

- 说「要一张配图」→ 单张横版封面触发本 skill; 多张文中插图 → **post-illustration-images**
- 说「要信息图 / 知识卡片」→ 不触发, 本 skill 就是**防信息图化**的
- 说「只要 prompt」: 目标是封面图 → 本 skill 文字模式; 目标是写作或通用 prompt → 不触发
