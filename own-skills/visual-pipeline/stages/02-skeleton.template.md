## HTML 输出要求 (`02-skeleton.html`)

- **顶部明确声明**: "本页所有内容为模拟数据, 仅用于展示信息密度与排版"
- **灰盒**: 仅中性灰阶 (#fff / #f5f5f5 / #ccc / #888 / #444), 不上品牌色
- **每个区域** 标中文区域名
- **中间用真实模拟数据** (中文 + 真实结构, 不是 lorem, 不是 `[字段名]` 占位符)
- **旁边图例** 解释每条模拟数据对应哪个字段 (字段名来自显示信息段第 4 节)
- **多状态** 上下排列 (不用 tab 切换, 此阶段保持静态对照)
- 每个状态前用 H2 标题: "入口态 / 选中态 / 空数据态" 等
