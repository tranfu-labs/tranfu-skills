## HTML 输出要求

**每个候选 `03-alt-*.html` 必须含两个 section, 顺序固定**:

### Section 一: 组件样例库 (顶部)

- 按第 2 节组件清单, **每个组件类型逐一给样例**
- 多个变体并排展示 (例: 主按钮 / 次按钮 / 危险按钮 / 禁用 全部画出)
- **不绑定具体字段** (这里是设计参考, 不是页面)
- 每个组件样例旁标组件名 (与第 2 节对应)

### Section 二: 整体组合 (底部)

- 用 Section 一中的组件 + 显示信息段第 4 节**真字段**, 按显示框架段区域+状态**拼起来**
- 至少展示一个核心状态 (一般是选中态)
- **绑定真实字段** (与显示信息段一一对应)

### 选定版 `03-selected.html` 规则

- 起始 = `cp 03-alt-<选中调性>.html 03-selected.html`
- 后续**主要在 Section 二「整体组合」内**迭代到像素级
- **Section 一「组件样例库」保留**, 后期回看可验证是否漂移
- 改主意切候选: 见 `change-flow.md` 第四节
