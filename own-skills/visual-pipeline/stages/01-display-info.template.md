## 4. 字段清单 + 优先级 ⭐

(本节是显示框架段最直接的输入. 字段来自上游已确定的内容; 优先级**从场景+动线推导**, 不是拍.)

| 字段 | 数据源 | 量级 | 优先级 | 服务于哪些场景 |
|---|---|---|---|---|
| ... | ... | ... | 必看 | [场景一] 主, [场景二] 主 |
| ... | ... | ... | 重要 | [场景一] 次 |
| ... | ... | ... | 辅助 | [场景三] |

**优先级语义**:
- **必看**: 主场景必看, 必须放显著区域 (头部 / hero / 一目了然处)
- **重要**: 主场景需要但可放二级位置
- **辅助**: 次场景或辅助, 可折叠 / 隐藏
- **留档**: 不展示 (或仅 hover/click 露出)
