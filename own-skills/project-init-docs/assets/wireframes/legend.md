# 控件字符图例

画页面时一律照这里的画法，不自创、不画状态变体。字符图只回答"有哪些控件、在什么位置"，控件长什么样属于视觉层，不在此表达。
容器类必须按消歧义规则在顶边/行首标明身份（见 [AGENTS.md](AGENTS.md)）。

## 简单控件

```
按钮       [ 文字 ]
下拉       [ 文字        ▾ ]
输入框     [____________]
搜索框     [ 🔍 文字      ]
复选       ☐ 选项      ☑ 选项
单选       ◯ 选项      ◉ 选项
开关       ( ● )
文字块     ≈≈≈≈≈≈≈≈≈≈
图片/媒体  ▢      或   [图]
面包屑     首页 › 列表 › 详情
```

## 容器（必须标明身份）

卡片：

```
┌─ 卡片 ────────┐
│               │
└───────────────┘
```

表格：

```
┌─ 表格 ──────┬───────┬───────┐
│ 表头        │       │       │
├─────────────┼───────┼───────┤
│ 行          │       │       │
└─────────────┴───────┴───────┘
```

标签页（线性，加前缀，当前页用 ‹ › 括住）：

```
标签页 ‹标签A› 标签B  标签C
```

弹窗（浮层，双线框）：

```
╔═ 弹窗 ═══════╗
║             ║
╚═════════════╝
```

页面区块（顶边标明身份）：

```
┌─ 顶部导航 ─────────────────┐
└────────────────────────────┘
┌─ 侧栏 ──┐┌─ 主内容 ────────┐
│         ││                 │
└─────────┘└─────────────────┘
```

## 流转图节点与箭头（flow.md 专用）

页面节点（实线框，内部指向其 page.md）：

```
┌─ 首页·任务列表 / ─┐
│ → pages/home.md   │
└────────────────────┘
```

同页态节点（虚线框，无跳转的态变化）：

```
┌┄ 同页刷新，新任务入列 ┄┐
┊（无跳转）            ┊
└┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┘
```

跳转箭头（每条打编号，对应步骤表）：

```
横向：── ① 触发 ─▶     回边：◀── ② 返回 ──
纵向：│                ▼
```
