# 黑白线性图标风

一句话出一张极简黑白线性图标——透明底、粗黑圆角线条、单主体居中、不带文字, 也可以只要 Prompt。

## 什么时候用它

**主题描述**:

我脑子里有个主题 (AI 工作流 / 目标拆解 / 多轮评审 / 数据分析), 想要一个抽象符号一眼表达它。

**UI 入口**:

我在铺产品页面, 需要功能入口图标 / 空状态图标 / 小辅助符号, 缩到 24-48 像素也要看得清。

**辅助符号**:

我想要一个近似 Logo 感的模块标识或辅助图形, 但不用做到正式品牌 Logo 定稿那一步。

**只要文案**:

我要的是图标生成 Prompt 本身 (中文 / 英文 / 附负面词), 拿去别处用。跟它说「只要 Prompt / 不出图 / 先给方案」就停在文字, 不动图像生成。

**不接**:

彩色插画 / 写实 / 3D 渲染 → 换别的工作流; 带标题的文章封面 → **article-cover-image**; 带文字的信息图 / 知识卡片 → 专门的信息图工作流; 正式品牌 Logo 定稿 / 商标 / VI 主标识 → Logo 设计工作流或人工设计。

## 它会产出什么

**默认直接出图, 不先给方案也不只讲 Prompt——除非你明确说「只要 Prompt / 不生成图 / 先分析」才停在文字。**

- **图像**: 一张黑白线性图标, 透明底 (不支持透明就用纯白), 粗黑圆角线条, 单主体居中, 无文字 / 字母 / 数字
- **出图前自检**: 7 项 Prompt 自检, 必须能从 Prompt 里直接读到「黑白配色 / 透明底 / 粗黑圆角线条 / 单主体居中 / 禁止文字 / 缩到 24-48 像素可辨 / 附了负面词」再放行
- **出图后验成品**: 7 项成品验证, 任一项不过就回去收紧 Prompt 重出
- **失败退档**: 同一类失败连出两次自动退到更简单的单主体几何符号再试一次, 不无限重跑
- **只要 Prompt 模式**: 你明说要 Prompt, 返回中文 Prompt + 英文 Prompt + 负面词清单, 不调用图像生成
- **绝不会做**: 出彩色 / 加阴影渐变 3D / 出海报 / 出带文字的信息图 / 交付正式品牌 Logo

## 前置条件与边界

**前置**:

运行时要有图像生成能力 (Codex 或 Claude Code 的出图工具)。你要给一个主题 / 概念 / 使用场景, 没有就先问你。

**相邻 skill 分工**:

| 任务 | 交给 |
|---|---|
| 带标题的文章封面 / 横版图 | **article-cover-image** |
| 正式品牌 Logo 定稿 / 商标 / VI 主标识 | Logo 设计工作流或人工设计 |
| 带文字的信息图 / 知识卡片 | 专门信息图工作流 |

**不接的场景**:

- 彩色插画 / 写实 / 摄影风 / 3D / C4D 立体渲染
- 复杂海报 / 带标题横版图 / 文章封面
- 信息图 / 知识卡片 / 流程说明图 (带文字标签或图表)
- 品牌正式 Logo / 商标 / VI 主标识

**微妙边界**:

- 「logo-like」辅助符号 / 概念标记 → 触发
- 「品牌 Logo 定稿 / 商标 / VI 主标识」 → 不触发, 转出去
- 「只要 Prompt」 且目标是黑白线性图标 → 只返 Prompt 文字, 不出图
- 「只要 Prompt」 但目标不是图标 → 不触发
