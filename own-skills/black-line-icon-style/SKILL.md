# Skill：极简黑白线性图标风

## 适用场景
当用户需要生成 icon、符号、logo-like 图形、科技感图标、抽象概念图、UI 功能入口图标、空状态插画时，使用该风格。

尤其适合：
- Prompt 多轮评审
- 目标驱动拆解
- AI 工作流
- 多步骤流程
- 任务分解
- 审核 / 反馈 / 评分
- 数据分析 / 结果确认
- 系统结构 / 流程编排

---

## 风格目标
生成一种极简、现代、抽象、黑白线性图标风格的视觉图形。

整体应像手绘科技符号，但保持干净、克制、几何化，适合用于 UI、产品、品牌辅助图形或视觉系统。

---

## 视觉特征
- 白色、透明或极浅灰背景
- 使用纯黑色粗线条
- 线条圆角、流畅、连续
- 主体居中，构图简洁
- 强识别度，强轮廓感
- 抽象科技感或系统符号感
- 使用少量几何曲线或基础图形表达概念
- 图形应像现代 UI icon，而不是复杂插画

---

## 生成规则
生成图像时，应遵循以下规则：

1. 使用黑白配色，不使用彩色。
2. 使用粗黑色圆角线条。
3. 背景优先使用透明底；如不支持透明底，则使用纯白背景。
4. 主体居中，留出充足留白。
5. 线条应流畅、有轻微手绘感，但不能杂乱。
6. 图形应简洁、抽象、现代。
7. 使用图形符号表达概念，不依赖文字。
8. 不要加入复杂装饰。
9. 不要使用阴影、渐变、3D、纹理。
10. 不要写实化。
11. 不要出现文字、字母、数字，除非用户明确要求。
12. icon 应适合缩小后在 UI 中使用。

## 主题生成模板

### 目标驱动拆解 图标
生成一个“目标驱动拆解”的极简黑白线性图标。透明背景，粗黑色圆角线条。上方是一个靶心或目标符号，中心有箭头命中目标；下方用三到四个节点表示目标被拆解成任务、数据和结果。节点之间用简洁连线连接，整体像一个清晰的目标分解结构图。主体居中，现代 UI icon 风格，无文字、无阴影、无渐变、无复杂细节。

### Prompt 多轮评审 图标
生成一个“Prompt 多轮评审”的极简黑白线性图标。透明背景，粗黑色圆角线条。中心可以是一个文档或提示词卡片，周围用循环箭头或多个阶段节点表示多轮迭代评审。节点可用对话气泡、勾选、星标或反馈符号表达“反馈、修改、确认、评分”。主体居中，构图简洁，现代 UI icon 风格，无文字、无阴影、无渐变、无复杂细节。

### AI 工作流 图标
生成一个“AI 工作流”的极简黑白线性图标。透明背景，粗黑色圆角线条。使用节点、连接线、箭头和一个抽象 AI 核心符号表现输入、处理、输出。主体居中，构图简洁，现代 UI icon 风格，无文字、无阴影、无渐变、无复杂细节。
