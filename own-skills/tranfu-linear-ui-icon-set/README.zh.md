# 线性 UI 图标组

让 AI 一次画出 4 个风格统一的 64×64 极简线性 UI 图标, 直接能塞进界面里用。

## 什么时候用它

**为界面配图标**:

我在做一个 Web 后台或 App 页面, 缺一组风格统一的功能图标, 想让 AI 一次给我 4 个候选。

**说明产品用途**:

我用一句话说明用途 (支付 / 云盘 / 消息 / 日历), 让 AI 顺着场景挑合适的图形隐喻, 不用我一个个描述形状。

**延续现有图标**:

界面已经在用一套柔和色 + 6px 圆角描边的图标, 新增图标要延续同一套视觉语言, 不能突然换风格。

**指定图标配色**:

我只要蓝色和绿色两色, 或者只要橙紫两色, 让整套图标围绕限定的主题色展开。

**指定尺寸和线宽**:

我已经在 64×64 的正方形直角底板上排版, 别的画布尺寸不接。

**不接**:

- 品牌 logo / 品牌标识 → 那是品牌设计工具的活
- 整页 UI 稿 / 海报 → 走 **ui-ux-pro-max**
- 架构图 / 流程图 → 走 **fireworks-tech-graph**
- 3D / 拟物 / 卡通 / 手绘涂鸦 / 像素风

## 它会产出什么

**只出 4 个独立方形图标, 从不合成海报也不做整页 UI**——最要紧的一点。

- **画布**: 4 张 64×64px 直角正方形底板, 主体居中, 尺寸控制在 32-40px, 四周留白充足
- **描边**: 6px 均匀线性描边, 圆角线帽 + 圆角转折, 四个图标视觉重量一致
- **配色**: 每个图标一个柔和浅色底 + 一个更深主题线条色, 默认候选是 `#FFF3E8` 橙 / `#F1EAFE` 紫 / `#EAF2FF` 蓝 / `#EAF8F2` 绿
- **图形方案**: 4 个方案彼此不同 (不同隐喻或结构), 但描边 / 比例 / 留白 / 圆角处理保持统一
- **绝不会做**: 加文字 / 数字 / 字母 / 水印 / logo; 用渐变 / 阴影 / 高光 / 纹理; 出圆角底板或透明背景

## 前置条件 / 边界

**前置**:

有一个能按提示词生图的 AI (Midjourney / DALL·E / Nano-banana 等), 本 skill 只出提示词, 不自己渲染像素。

**相邻 skill 分工**:

| 需求 | 交给 |
|---|---|
| 整页 UI 稿 / 海报 / 落地页视觉 | **ui-ux-pro-max** |
| 架构图 / 流程图 / 数据流图 | **fireworks-tech-graph** |
| 品牌 logo / 商标 / 品牌标识 | 不接, 用专门的品牌设计工具 |

**不接的场景**:

- 品牌 logo / 品牌标识 / 商标类需求
- 整页 UI / 海报 / 带文案的展示图
- 3D / 拟物 / 卡通 / 手绘涂鸦 / 像素风
- 透明背景 / 圆角底板 / 复杂容器

**微妙边界**:

- 想要 4 个以外的数量 (3 个 / 6 个 / 8 个) → 不触发, 硬约束就是 4 个
- 想要非 64×64 的画布 (128×128 / 32×32) → 不触发, 尺寸是硬约束的一部分
- 想要同色系里两个不同饱和度的色 → 触发, 保持一色一图的边界即可
