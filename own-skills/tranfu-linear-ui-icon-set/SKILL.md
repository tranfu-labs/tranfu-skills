# 线性 UI 图标组生成 Skill

## Skill 目的

生成一组由 **4 个独立图标** 组成的极简线性 UI 图标。\
四个图标需要保持统一的视觉规范，同时每个图标使用不同的图形方案或视觉隐喻。

## 适用场景

当用户需要生成、设计或描述以下内容时，使用本 Skill：

- 极简线性 UI 图标
- 小型图标组
- 64×64px 图标
- 适合界面使用的轻量化图标
- 带柔和浅色底板的现代图标
- 统一视觉规范的 icon set

## 输出要求

生成 **4 个独立图标**。

每个图标都必须位于独立的 **64×64px 正方形画板** 中。

四个图标需要独立展示，不要合并成海报、UI 页面、复杂插画或整体场景。

画面中不得出现任何文字、标题、字母、数字、标签、说明性符号、水印、Logo 或品牌标识。

## 画布规范

每个图标使用以下画布规范：

- 画布尺寸：64×64px
- 底板形状：正方形
- 底板边角：直角
- 不要圆角底板
- 不要透明背景
- 图标主体尺寸：约 32–40px
- 图标主体位置：居中放置
- 留白：四周留白充足

## 视觉风格

图标整体风格应为：

- 极简
- 线性
- 现代
- 清爽
- 轻量化
- 适合 UI 界面使用
- 简洁且易识别

四个图标必须保持统一的视觉语言，包括：

- 描边粗细
- 图形比例
- 留白关系
- 圆角处理
- 视觉重量
- 简洁程度

## 线条规范

图标只能使用线性描边表现。

描边规范如下：

- 描边粗细：6px
- 线帽：圆角线帽
- 转折：圆角转折
- 线条粗细：始终均匀一致
- 视觉重量：四个图标保持一致

不得使用不同粗细的线条。\
不得使用尖角线帽、方形线帽或毛边线条。\
不得使用复杂填充形状。

## 颜色规范

每个图标使用一个柔和浅色背景和一个更深的对应主题线条色。

推荐背景色：

- 柔和橙色背景：#FFF3E8
- 柔和紫色背景：#F1EAFE
- 柔和蓝色背景：#EAF2FF
- 柔和绿色背景：#EAF8F2

推荐线条色：

- 橙色主题线条
- 紫色主题线条
- 蓝色主题线条
- 绿色主题线条

每个图标最多只能使用 **1 个主题线条色**。

整体配色应保持柔和、干净、统一，避免过度饱和或刺眼。

## 构图规范

每个图标主体应满足以下要求：

- 简洁居中
- 清晰易识别
- 构图平衡
- 不拥挤
- 不使用复杂细节
- 不添加过多装饰元素

四个图标的图形方案应彼此不同，但在视觉语言、描边、比例、留白和圆角处理上保持一致。

## 禁止内容

不得包含：

- 文字
- 标题
- 字母
- 数字
- 标签
- 说明性符号
- 水印
- Logo
- 品牌标识
- 3D 风格
- 拟物化风格
- 复杂插画风格
- 卡通风格
- 手绘涂鸦风格
- 像素风格
- 厚重装饰元素
- 阴影
- 渐变
- 高光
- 纹理
- 噪点
- 复杂背景
- 多层复杂细节
- 拥挤构图
- 过多小元素
- 不对称失衡
- 不同粗细线条
- 尖角线帽
- 方形线帽
- 毛边线条
- 圆角底板
- 透明背景
- 复杂容器
- 过多颜色
- 彩虹配色
- 强对比刺眼颜色
- 超出 64×64px 画布的元素
- 过大或过小的图标主体
- 成套海报
- UI 页面
- 带文案的展示图

## 生成提示词模板

生成 4个独立方案的极简线性 UI 图标。

每个图标位于独立的 64×64px 正方形画板中，底板为直角正方形，不要圆角底板。

四个图标应像同一套 UI icon set 中的成员，整体风格保持极简、线性、现代、清爽、轻量化，适合 UI 界面使用。

每个图标主体居中放置，主体尺寸控制在 32–40px 之间，四周保留充足留白。

图标仅使用线性描边表现。\
描边粗细为 6px。\
使用圆角线帽和圆角转折。\
保持线条粗细均匀，视觉重量一致。

每个图标使用一个柔和浅色背景和一个对应的更深主题线条色。\
可使用以下配色：

- \#FFF3E8 背景，搭配橙色线条
- \#F1EAFE 背景，搭配紫色线条
- \#EAF2FF 背景，搭配蓝色线条
- \#EAF8F2 背景，搭配绿色线条

每个图标最多使用 1 个主题线条色。

四个图标的图形方案需要彼此不同，但视觉语言、描边、比例、留白和圆角处理必须保持一致。

不要出现任何文字、标题、字母、数字、标签、说明性符号、水印、Logo 或品牌标识。

不要生成海报、UI 页面、复杂插画或四个图标合并成的整体场景。

避免 3D、拟物化、复杂插画、卡通风、手绘涂鸦风、像素风、阴影、渐变、高光、纹理、噪点、复杂背景、过多细节、拥挤构图、不同粗细线条、尖角线帽、方形线帽、毛边线条、圆角底板、透明背景、彩虹配色、强对比刺眼颜色或厚重装饰元素。
