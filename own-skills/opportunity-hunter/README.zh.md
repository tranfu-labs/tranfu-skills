# 创业机会猎手

把「找机会」从等灵感变成一套猎法——四猎场定位、why-now 扳机、卖铲子提问、四道杀闸快筛。

## 什么时候用它

**手头有观察**:

我最近反复被某件事烦到, 或者刷到一条新闻 / 政策 / 新模型能力, 隐约觉得像个机会, 想让它帮我判断是不是真机会。

**判断新催化剂**:

某个大厂刚下场 / 模型刚降价 / 新法规刚出, 我在想这是顺风还是逆风、是扳机还是丧钟, 想推清楚因果链再决定要不要下场。

**评估创业想法**:

我在琢磨做工具 / 平台 / 基础设施, 想让它帮我识别这个铲子是不是「挖到挖不到都得买」, 还是那种大厂免费送 / 靠涨价才赚钱的反向铲子。

**头脑风暴方向**:

我在想创业方向, 想让它带我从「想点子」切到「选站位」——去哪个猎场蹲, 对催化剂问哪句话。

**筛选已有想法**:

我已经攒了几个候选想法, 想用四道杀闸挨个杀, 只留下扛得住的那几个。

**不接**:

AI 方向存活候选的立项级死亡风险深度自检 → **ai-startup-feasibility-check**; 写商业计划书 / 融资材料 / 竞品分析 → 不触发。

## 它会产出什么

**结论只有两态: 存活或杀掉——绝不给「再想想」式的暧昧结论**。

- **猎场归位**: 把你的想法归到四猎场之一 (自己每天骂的摩擦 / 前沿能力位移 / 监管变动 / 无聊行业的脏活), 或直接指出它不属于任何一个
- **why-now 扳机检验**: 逼出「谁 + 现在 + 必须花钱买什么」三段式答案, 答不出就发警告
- **四道杀闸表**: 痛是真的 / 能防御大厂 / 杠杆方向 / 单位经济, 顺序杀, 前关挂了不看后关
- **薄切片建议** (仅存活候选): 一句话说清最细的一条真实链路是什么
- **AI 方向衔接**: 存活候选且落在 AI 领域, 主动提示走 **ai-startup-feasibility-check** 做深度自检
- **绝不会做**: 替你写商业计划书 / 融资材料 / 竞品报告

## 前置条件与边界

**前置**:

一个具体的候选 (观察 / 痛点 / 新闻 / 政策 / 新模型能力) 或明确的方向意图。空手来「帮我找个机会」→ 会先反问你所在的猎场。

**相邻 skill 分工**:

| 动作 | 交给 |
|---|---|
| AI 方向存活候选的深度死亡风险自检 | **ai-startup-feasibility-check** |

**不接的场景**:

- 写商业计划书 / 融资材料 / 竞品分析
- 已经决定要做的项目做技术选型
- 与找机会无关的行业趋势报告

**微妙边界**:

- 问「这个新变化是不是机会」→ 触发; 问「AI 会怎么改变某行业」这类泛趋势讨论 → 不触发
- 存活候选是 AI 方向 → 主动衔接 **ai-startup-feasibility-check**; 非 AI 方向 → 停在快筛结论
