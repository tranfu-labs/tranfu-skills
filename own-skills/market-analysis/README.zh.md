# market-analysis

一个把市场 / 行业 / 赛道放进 **12 维度并行搜索 + AI 原生分析框架 + 非对称机会矩阵 + 脆弱性审计**, 跑出结构化全景市场分析报告的 skill — 适合创业者 / 产品经理 / 分析师评估市场.

## 什么时候用它

- 想对一个市场 / 行业 / 赛道做**系统性全景分析**, 不是零散事实拼一起
- 需要**多维度覆盖** — 规模、竞争、用户、资本、政策一次跑齐
- 要做**竞争格局分析** / 机会点挖掘 / TAM / SAM / SOM 测算
- 想要一份能直接拿去写决策简报或投资摘要的结构化报告

## 怎么触发它

跟 Claude 说:

- "分析 AI 客服市场"
- "帮我做在线教育行业市场调研"
- "竞争格局分析: 新能源汽车赛道"
- "market analysis for XX"

提交时最好附上: **目标市场 / 行业关键词 + 地域范围**. 说"**快速看看**"切轻量模式 (6–8 次搜索), 说"**深入研究**"切深度模式 (20+ 次搜索), 默认走标准深度 (12–16 次搜索). 给具体市场名 (如"AI 客服 SaaS") 比泛词 (如"AI 市场") 效果好.

## 你会看到什么

一份 8 章结构化报告: 执行摘要 → 赛道全景 (含拐点信号 + TAM / SAM / SOM + 非对称机会矩阵) → 竞争格局 (含量化五力 + 竞品价值链拆解) → 用户需求洞察 (含 Top 5 痛点 + 蓝海需求) → 投融资与政策 → 机会洞察 (3–5 个具体机会含 MVP 路径) → 脆弱性审计 → 战略建议. 每个关键数据点标注来源与时间. 依赖 web search 拿实时数据, 离线环境效果打折.

## 相邻 skill

| 想干的事 | 交给 |
|---|---|
| 用研究员心智做深度赛道研究, 带反共识洞察和质量闸门 | **elite-market-researcher** (设计上与本 skill 配合: 搜索流程 vs 研究心智) |
| AI 产品完整商业可行性打分 (PEST → 五力 → SWOT → BMC, 120 分制) | **business-analysis-pipeline** |
| 只要竞品情报 + SWOT, 不要全景市场分析 | competitive-analyst (VoltAgent, 外部) |

**不接的场景**: 财务建模和可行性打分 (走 business-analysis-pipeline); 只做单一竞品 SWOT 对比 (更轻的工具够用); 纯英文市场且无中国视角 — 搜索词模板偏中文, 关键词需微调.
