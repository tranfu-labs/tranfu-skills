# TranFu Website Design Skill

This package contains the TranFu website design skill with bundled Figma logo assets.

## Install

Place the whole `tranfu-website-design` folder at:

```text
~/.codex/skills/tranfu-website-design/
```

The installed folder must contain:

- `SKILL.md`
- `references/`
- `assets/`
- `agents/`

## Included Logo Assets

- `assets/logo-lockup-h-en-primary-on-light.svg`
- `assets/logo-symbol-primary.svg`
- `assets/logo-wordmark-primary.svg`
- `assets/app-icon-primary.svg`

These assets were extracted from the TranFu Figma file and must be copied into target projects under `public/brand/` before use.

Do not reference local Downloads paths or old deployed-site logo assets. Do not redraw, recolor, stretch, crop, or add effects to the logo assets.
