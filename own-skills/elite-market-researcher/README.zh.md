# elite-market-researcher

用 McKinsey 合伙人 + 投行首席 + VC 合伙人 + 行业操盘手的复合心智做市场 / 赛道 / 行业深度研究, 输出带反共识洞察和事前验尸的完整报告 — 适合创业方向评估、投资判断、战略入场决策的严肃场景.

## 什么时候用它

- 要对一个赛道 / 行业做**严肃的方向评估或投资判断**, 不只是浏览资料
- 需要一份**超越"框架填空"** 的深度市场研究报告, 有观点、有数字、有立场
- 做创业方向评估、竞争格局分析、行业研报, 决策要压亿元级筹码
- 想拿到一份**领先行业 3-5 年视野** 的判断, 含反共识观点和脆弱性审计

## 怎么触发它

跟 Claude 说:

- "分析 AI 代码审查赛道"
- "帮我看看 Agent 中间件市场"
- "以 VC 视角评估这个创业方向"
- "做一份 XX 行业的深度研报"

提交时最好附上: **目标赛道 / 行业名称** + **决策场景** (投资 / 创业 / 战略入场) (必填); 关注的地域范围、时间窗口、深度要求 (选填, skill 会自动选合适模式).

## 你会看到什么

- **10 章结构化报告**: 执行摘要 → 研究边界 → 赛道全景 (含拐点信号) → 竞争格局 (含 unit economics) → 用户洞察 → 投融资与政策 → 机会洞察 → 反共识观点 → 脆弱性审计 (事前验尸) → 战略建议
- **每个关键数字带来源 + 时间**, 每个判断**三层标注** (事实 / 推断 / 观点), 拒绝"形容词分析"
- **反共识章节 + 事前验尸** 是本 skill 的差异化交付, 强制识别至少 3 个拐点信号 + 2 个非共识判断 + 各项风险的触发条件
- 输出前过 **10 关质量闸门自检**, 不达标返工不交付

## 相邻 skill

| 想干的事 | 交给 |
|---|---|
| AI 产品的商业可行性 / 投资尽调 / 商业分析全流程 | **business-analysis-pipeline** |
| 12 维度并行搜索 + 标准市场分析报告 (无反共识心智) | **market-analysis** |
| AI 创业方向快速可行性判断 (不跑深度研究) | **ai-startup-feasibility-check** |
| 一个赛道里挖机会点 / 排序机会 | **opportunity-hunter** / **ai-opportunity-evaluation** |

**不接的场景**: 只要一次搜索答一个具体问题 (走普通对话就行); 写正式的 pitch deck / 商业计划书 (那是下游文档产出); 跟踪已上线产品的 KPI (那是运营分析, 不是市场研究); 只想"了解一下"某个市场 (本 skill 的深度和长度都不匹配, 用 market-analysis 更合适).
