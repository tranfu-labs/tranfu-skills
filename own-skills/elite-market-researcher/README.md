# elite-market-researcher

> 顶尖市场研究员心智系统 — 让 AI 以 McKinsey 合伙人 + 投行首席 + 顶级 VC + 行业操盘手的复合视角，输出具备反共识洞察、可支撑亿元级决策的深度市场研究报告。

## 什么时候用它

当需要对一个市场 / 行业 / 赛道做**严肃决策依据**的研究时触发。典型触发短语：

- "做市场分析 / 分析 XX 赛道 / 研究 XX 行业"
- "竞争格局分析 / 投资机会判断 / 创业方向评估"
- "market research / industry analysis / 赛道判断"
- "做行业研报 / 帮我看看 XX 市场"

**不适用**：已有产品的商业可行性评估（走 `business-analysis-pipeline`）、纯项目打分（走 `project-scoring`）、功能规划 spec（走 `write-spec`）。

## 使用技巧

> 由 tranfu-publish 引导起草, 作者签字.

### 材料方案
- 可配合 `market-analysis` skill 使用（本 skill 负责心智，market-analysis 负责搜索流程）
- 单独使用时走内置简化搜索流程，不依赖外部 skill

### 推荐用法
- 首次使用建议说"深入研究 XX 赛道"触发深度模式（20+ 次搜索）
- 可指定视角切换：说"VC 视角"强化投资判断，说"创业视角"强化 MVP 路径
- 关键边界（地域、子赛道、决策场景）越清晰，输出越精准

### 已知限制
- 数据时效依赖搜索引擎结果，非实时数据库
- 报告结构固定 10 章节，不支持自定义章节裁剪
- 单次输出较长，快速模式也需完整跑六大原则 + 10 关自检
