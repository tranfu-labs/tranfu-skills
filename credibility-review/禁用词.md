## 13. 鼓励词白名单（不要误改）

以下词在 § 1–12 grep 命中后**不计违规**——它们是诚实 / 承认错误的标志：

```
我觉得 我猜 大概是
先这么扛着 暂时没修 还没解决
我以为 猜错了 没意识到 漏看了 想当然 写糊了 骂了一句
```

**对照规则**：
- "应该 / 想必 / 理论上"——禁止（含糊的标志）。
- "我觉得 / 我猜 / 大概是"——鼓励（诚实的标志）。
- 两者意思相近但语气意图不同。区分时按 SKILL.md Step 4.6 LLM 判读"作者承不承认自己不确定"。

## 英文文章 — §14 英文营销词（必改）

英文文章的等价营销 / 升华虚词。比中文 list 谨慎得多——下面这些词在严肃技术写作里几乎不会出现，命中即营销腔信号。

```
revolutionary  groundbreaking  game-changer  game-changing  cutting-edge
state-of-the-art  next-generation  paradigm shift  unleash  unleashing
supercharge  supercharged  democratize  seamlessly  effortlessly
unprecedented  world-class  industry-leading  best-in-class
transformative  unparalleled  ushering in (a new era)
```

**白名单 / 不要误改**：
- `state-of-the-art` 出现在论文 abstract / benchmark 比较段时（"achieves state-of-the-art on MMLU"）→ OK，是研究术语不是营销词。命中后 LLM 判定语境。
- `transform` 单独出现描述实际算法（"transform tokens into embeddings"）→ OK；`transformative impact` / `transform the industry` → 命中。

## §15 英文 hero / PR 句式（必改 / 部分退稿）

```
this is just the beginning      what's next            stay tuned
paving the way                  ushering in            our mission
our vision                      our north star         doubled down
we (have )?always (believed|known)
the journey (is just|continues|has only just)
we couldn't be more (excited|proud)
```

文末出现"this is just the beginning / what's next / stay tuned" → **退稿**（与中文 §12 CTA 等价）。

正文出现 "our mission / our north star / our vision" 等抽象使命句、未挂载具体物 → **必改**（与 §10 隐蔽 hero 同款）。

## §16 英文 CTA / 文末签名（**退稿**）

```
Contact us       Learn more       Get started        Sign up
Try it now       Get in touch     Reach out          Subscribe (now|here|to)
Click here       Visit our        Follow us
buy ... at       order now        available at       on my (art )?store
```

任一以独立行 / footer 形态出现 → **退稿**。

**白名单**：研究博客的 "feedback@anthropic.com" / "reach out to <email>" 在 acknowledgments / footnotes 里属于业务联系方式而不是营销 CTA——LLM 看上下文，靠近 acknowledgments / closing 的视为正常，靠近 product pitch 的视为 CTA。

## §17 英文含糊时间（必改 soft）

```
recently        a while back        some time ago       not long ago
in the past few (months|weeks)      back in the day
the other day（口语，但仍 vague）
```

命中后 LLM 看是否有附加时间锚点（具体日期 / 月份 / commit hash / version 号）；有锚点 → 不计违规，无锚点 → **必改**。

## 英文聚合 grep（独立于中文 grep，可并行跑）

```bash
grep -nEi "revolutionary|groundbreaking|game-?changer|cutting-edge|state-of-the-art|next-generation|paradigm shift|unleash|supercharge|democratize|seamlessly|effortlessly|unprecedented|world-class|industry-leading|best-in-class|transformative|unparalleled|ushering in|this is just the beginning|what'?s next|stay tuned|paving the way|our (mission|vision|north star)|doubled down|we (have )?always (believed|known)|contact us|learn more|get started|sign up|try it now|get in touch|reach out|subscribe (now|here|to)|click here|visit our|follow us|buy .* at|order now|available at|on my( art)? store" <target>
```

**注意**：`-i` 是英文版关键参数，缺了会漏 "Revolutionary" 之类首字母大写命中。中文 grep 不需要 `-i`。

每命中 1 行 → 候选违规：CTA 类 = 退稿，hero 文末类 = 退稿，营销虚词 / hero 抽象使命 = 必改。LLM 判读语境（§14–§16 各列了白名单条件）。
